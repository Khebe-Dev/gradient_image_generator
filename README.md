A Python command-line tool that generates a 24-bit BMP image with a colour gradient between two user-defined points.

## Usage
python gradient.py 10,20=10,50,255 90,80=60,20,10 -o output.bmp

## Features

Accepts two points with RGB values

Interpolates colours across the image

Writes a BMP file without external libraries
